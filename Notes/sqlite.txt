pi@rasp-test:~ $ sqlite3 test.db

SQLite version 3.34.1 2021-01-20 14:10:07
Enter ".help" for usage hints.
sqlite> CREATE TABLE weather(id INTEGER PRIMARY KEY AUTOINCREMENT, temperature REAL, humidity REAL, date DATETIME);
sqlite> .tables
weather
sqlite> INSERT INTO weather (temperature, humidity, date) VALUES (20.2, 60, datetime("now"));
sqlite> INSERT INTO weather (temperature, humidity, date) VALUES (21, 70, datetime("now"));
sqlite> select * from weather;
